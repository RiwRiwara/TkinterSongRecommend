# Setimg program

## Requirements:
- Python 3.x installed on the system.
- virtual env 
```
pip install virtualenv
```
### Get start
- Download and Extract file.
- open file directory `/songProject/`.
- open terminal cmd or powwershell.

### Run code in terminal
```
.\venv\Scripts\activate
```
### Start Program
```
cd .\program\
```
```
python .\main.py
```
